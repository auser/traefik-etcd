<h1>Traefik Config Generator with etcd</h1> <p>This is a tool to generate Traefik configuration for a given set of hosts.</p> <h2>Usage</h2> <pre style="background-color:#263238;"><span style="color:#82aaff;">cargo run</span><span style="color:#89ddff;"> --</span><span style="color:#f78c6c;">bin</span><span style="color:#82aaff;"> traefik-config-generator</span><span style="color:#89ddff;"> --</span><span style="color:#f78c6c;">config</span><span style="color:#82aaff;"> config/config.yml </span></pre> <h2>Configuration</h2> <p>The configuration is done in the <code>config/config.yml</code> file.</p> <h3>Hosts</h3> <p>Each host has a domain, a list of paths, and a list of deployments.</p> <h4>Paths</h4> <p>Each path has a path, a list of deployments, a list of middlewares, and a boolean to strip the prefix. The deployments are keyed by the deployment name, which is used to determine which router to use.</p> <h4>Deployments</h4> <p>Each deployment has an ip, a port, a weight, and a boolean to determine if
